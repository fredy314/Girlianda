# 🎄 Girlianda (Гірлянда)

Проєкт контролера **двох незалежних** двоканальних LED гірлянд на базі ESP32. Кожна гірлянда має власні режими, швидкість та яскравість. Забезпечує зручне керування через сучасний веб-інтерфейс або API. Оснащений інтелектуальною системою підключення до WiFi та можливістю оновлення прошивки через браузер (OTA).

## ✨ Можливості

### 🎨 Режими світіння
Кожна гірлянда підтримує 5 режимів:
- **0 - Постійне світіння**: Обидва канали світяться з однаковою яскравістю (AC режим).
- **1 - Почергове плавне**: Синусоїдальна зміна з перемиканням між каналами.
- **2 - Дихання**: Плавна синусоїдальна зміна яскравості (синхронно обидва канали).
- **3 - Хаос**: Випадкова зміна яскравості для створення "живого" ефекту.
- **4 - Свічка**: Ефект свічки з теплою пульсацією (70% база + 30% шум).

**Примітка**: Яскравість 0 = вимкнено.

### 🌐 WiFi та Мережа
Автоматичне підключення до WiFi через бібліотеку **WiFiManager**:
- Підтримка декількох мереж з автоматичним вибором найкращої
- Налаштування через файл `secrets.h`
- Опціональний статичний IP або DHCP
- mDNS підтримка (доступ через `girlianda.local`)
- Fallback до режиму Access Point при відсутності мереж

### 📱 Веб-інтерфейс
Сучасний, адаптивний веб-інтерфейс з темною темою:
- Кнопки швидкого вибору режимів
- Слайдери для регулювання швидкості анімації та яскравості
- Відображення статусу підключення в реальному часі
- Кнопка для OTA оновлення прошивки

### 🔄 OTA Оновлення
Оновлення прошивки через браузер без USB кабелю:
- Захищений доступ паролем до сторінки `/update`
- Інтеграція з **ElegantOTA**
- Зручний веб-інтерфейс для завантаження нової прошивки

### 🔌 API та Інтеграція
Повний контроль через HTTP GET запити для інтеграції з системами розумного будинку.

## 🔧 Технічні деталі

### Підключення
**Гірлянда A:**
- **Pin A1**: GPIO 10
- **Pin A2**: GPIO 7

**Гірлянда B:**
- **Pin B1**: GPIO 6
- **Pin B2**: GPIO 5

**Параметри PWM:**
- **Частота**: 1 кГц
- **Роздільна здатність**: 8 біт (0-255)
- **H-Bridge режими**: OFF, POS, NEG, AC (антипаралельне включення)

**Драйвер:** Mini-L298N H-Bridge (двоканальний драйвер двигунів)

**Схема підключення:**
- **Гірлянда A**: `GPIO 10`, `GPIO 7` → перший H-міст (OUT1, OUT2)
- **Гірлянда B**: `GPIO 6`, `GPIO 5` → другий H-міст (OUT3, OUT4)

> **Статус:** H-міст припаяний, прошивка підтримує дві незалежні гірлянди. Залишилось підключити другу LED гірлянду до виходів OUT3/OUT4.

## 🚀 Встановлення

1.  **Встановіть залежності**:
    Переконайтеся, що у вашому середовищі (Arduino IDE / PlatformIO) встановлені бібліотеки:
    -   `ESPAsyncWebServer`
    -   `AsyncTCP`
    -   `ElegantOTA`
    -   `WiFiManager` (для ESP32)
    -   `AuthenticationMiddleware`
    -   `RemoteXY` (для Bluetooth керування)
    -   ESP32 Board package

2.  **Налаштування**:
    Скопіюйте `secrets.h.example` в `secrets.h` та налаштуйте:
    ```cpp
    inline WiFiNetwork wifiNetworks[] = {
      {"YOUR_WIFI_SSID", "YOUR_PASSWORD"},
      {"YOUR_EXT_WIFI_SSID", "YOUR_PASSWORD"} // Опціонально
    };
    
    #define WIFI_HOSTNAME "girlianda"  // Опціонально
    #define AUTH_PASSWORD "your-ota-password"  // Для /update
    ```

3.  **Завантаження**:
    Скомпілюйте та завантажте скетч у вашу плату ESP32.

## 📖 Використання

Після запуску відкрийте Serial Monitor (швидкість **115200**).
Ви побачите IP адресу пристрою (наприклад, `192.168.1.105`) або адресу точки доступу (`192.168.4.1`).

Відкрийте цю адресу у браузері для доступу до панелі керування.

### 📱 Bluetooth Керування (RemoteXY)

Проєкт підтримує керування через Bluetooth Low Energy (BLE) за допомогою додатку **RemoteXY**.

**Підключення:**
1. Встановіть додаток **RemoteXY** на Android або iOS
2. У додатку натисніть "+" → "Bluetooth LE"
3. Знайдіть пристрій **"Girlianda"** та підключіться

**Елементи керування:**
- 🔀 **Перемикач гірлянди** - вибір між Garland A та Garland B
- 🎭 **Перемикач режиму** - вибір режиму (0-4)
- ⚡ **Слайдер швидкості** - регулювання швидкості анімації (0-100)
- 💡 **Слайдер яскравості** - регулювання яскравості (0-100)

**Примітка:** ESP32-C3 підтримує тільки BLE (Bluetooth Low Energy), не класичний Bluetooth.

### 📡 API

### Endpoints
| Метод | Endpoint | Опис |
|-------|----------|------|
| GET | `/` | Головна сторінка з веб-інтерфейсом |
| GET | `/api/status` | Отримати поточний стан обох гірлянд |
| GET | `/api/set?garland=A&mode=X` | Встановити режим `X` (0-4) для гірлянди A |
| GET | `/api/set?garland=B&speed=X` | Встановити швидкість `X` (1-100) для гірлянди B |
| GET | `/api/set?garland=A&brightness=X`| Встановити яскравість `X` (0-255) для гірлянди A |

**Примітка**: Параметр `garland` обов'язковий і може бути `A` або `B`.

### 🔐 OTA Оновлення
Для оновлення прошивки через браузер:
1.  Відкрийте `http://<IP-адреса>/update` або `http://girlianda.local/update`
2.  Введіть пароль (налаштований в `secrets.h`)
3.  Виберіть файл прошивки (`.bin`) та натисніть "Update"
