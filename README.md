# 🎄 Girlianda (Гірлянда)

Проєкт контролера двоканальної LED гірлянди на базі ESP32. Забезпечує зручне керування режимами світіння через сучасний веб-інтерфейс, серіал-порт або API. Оснащений інтелектуальною системою підключення до WiFi.

## ✨ Можливості

### 🎨 Режими роботи
Контролер підтримує 6 режимів роботи:
- **0 - Вимкнено**: Гірлянда вимкнена.
- **1 - Постійне світло**: Статичне світіння з можливістю регулювання яскравості.
- **2 - Протифаза (Alternating)**: Плавне перемикання між двома каналами.
- **3 - Синхронне дихання**: Обидва канали плавно розпалюються та згасають синхронно або у протифазі (залежно від реалізації).
- **4 - Хаос**: Випадкова зміна яскравості для створення "живого" ефекту.
- **5 - Мерехтіння**: Ефект свічки з теплою пульсацією.

### 🌐 Smart WiFi Connection
Інтелектуальна система підключення забезпечує надійний зв'язок:
1. **Сканування ефіру**: Пошук доступних мереж перед підключенням.
2. **Вибір найкращої мережі**: Автоматичний вибір між основною (`ssid`) та додатковою (`ssidExt`) мережею на основі рівня сигналу (RSSI).
3. **Адаптивна потужність**: Якщо підключення не вдається, контролер автоматично підвищує потужність передавача WiFi (від 8.5dBm до 15dBm).
4. **Failover AP**: У разі невдачі всіх спроб автоматично створюється Точка Доступу `Girlianda-AP`.

### 📱 Веб-інтерфейс
Сучасний, адаптивний веб-інтерфейс з темною темою:
- Кнопки швидкого вибору режимів.
- Слайдери для регулювання швидкості анімації та яскравості.
- Відображення статусу підключення в реальному часі.

### 🔌 API та Інтеграція
Повний контроль через HTTP GET запити для інтеграції з системами розумного будинку.

## 🛠 Обладнання та Підключення

- **Мікроконтролер**: ESP32 (DevKit V1 або аналог).
- **Драйвер**: Модуль **Mini-L298N H-Bridge** (двоканальний драйвер двигунів).
- **Схема підключення**:
  - **Гірлянда 1 (Канал А/B)**: `GPIO 10`, `GPIO 7` (використовується перший H-міст).
  - **Гірлянда 2 (Резерв/Майбутнє)**: `GPIO 5`, `GPIO 6` (використовується другий H-міст).
  
> **Примітка:** Апаратно передбачено можливість підключення другої незалежної гірлянди до пінів 5 та 6. У майбутніх версіях прошивки планується реалізація повного роздільного керування обома гірляндами.

## 🚀 Встановлення

1. **Встановіть залежності**:
   Переконайтеся, що у вашому середовищі (Arduino IDE / PlatformIO) встановлені бібліотеки:
   - `ESPAsyncWebServer`
   - `AsyncTCP`
   - ESP32 Board package

2. **Налаштування**:
   Відкрийте файл `Girlianda.ino` та внесіть свої WiFi дані:
   ```cpp
   const char* ssid = "YOUR_WIFI_SSID";
   const char* ssidExt = "YOUR_EXT_WIFI_SSID"; // Опціонально
   const char* password = "YOUR_WIFI_PASSWORD";
   ```

3. **Завантаження**:
   Скомпілюйте та завантажте скетч у вашу плату ESP32.

## 📖 Використання

Після запуску відкрийте Serial Monitor (швидкість **115200**).
Ви побачите IP адресу пристрою (наприклад, `192.168.1.105`) або адресу точки доступу (`192.168.4.1`).

Відкрийте цю адресу у браузері для доступу до панелі керування.

### 📡 HTTP API
| Метод | URL | Опис |
|-------|-----|------|
| GET | `/api/status` | Повертає JSON зі станом: `{"mode":1,"speed":50,"brightness":255}` |
| GET | `/api/set?mode=X` | Встановити режим `X` (0-5) |
| GET | `/api/set?speed=X` | Встановити швидкість `X` (1-100) |
| GET | `/api/set?brightness=X`| Встановити яскравість `X` (0-255) |

### 💻 Serial Команди
Ви можете керувати пристроєм через USB (Serial):
- `m=X` - Встановити режим (наприклад, `m=1`)
- `s=X` - Встановити швидкість (`s=100`)
- `b=X` - Встановити яскравість (`b=250`)
- `wifi` - Перезапустити сканування та підключення до WiFi
- `ap` - Примусово запустити режим Точки Доступу
- `scan` - Сканувати ефір та вивести список мереж
- `status` - Показати поточний статус та IP
