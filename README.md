# 🎄 Girlianda (Гірлянда)

Проєкт контролера двоканальної LED гірлянди на базі ESP32. Забезпечує зручне керування режимами світіння через сучасний веб-інтерфейс або API. Оснащений інтелектуальною системою підключення до WiFi та можливістю оновлення прошивки через браузер (OTA).

## ✨ Можливості

### 🎨 Режими роботи
Контролер підтримує 6 режимів роботи:
- **0 - Вимкнено**: Гірлянда вимкнена.
- **1 - Постійне світло**: Статичне світіння з можливістю регулювання яскравості.
- **2 - Протифаза (Alternating)**: Плавне перемикання між двома каналами.
- **3 - Синхронне дихання**: Обидва канали плавно розпалюються та згасають синхронно або у протифазі (залежно від реалізації).
- **4 - Хаос**: Випадкова зміна яскравості для створення "живого" ефекту.
- **5 - Мерехтіння**: Ефект свічки з теплою пульсацією.

### 🌐 WiFi та Мережа
Автоматичне підключення до WiFi через бібліотеку **WiFiManager**:
- Підтримка декількох мереж з автоматичним вибором найкращої
- Налаштування через файл `secrets.h`
- Опціональний статичний IP або DHCP
- mDNS підтримка (доступ через `girlianda.local`)
- Fallback до режиму Access Point при відсутності мереж

### 📱 Веб-інтерфейс
Сучасний, адаптивний веб-інтерфейс з темною темою:
- Кнопки швидкого вибору режимів
- Слайдери для регулювання швидкості анімації та яскравості
- Відображення статусу підключення в реальному часі
- Кнопка для OTA оновлення прошивки

### 🔄 OTA Оновлення
Оновлення прошивки через браузер без USB кабелю:
- Захищений доступ паролем до сторінки `/update`
- Інтеграція з **ElegantOTA**
- Зручний веб-інтерфейс для завантаження нової прошивки

### 🔌 API та Інтеграція
Повний контроль через HTTP GET запити для інтеграції з системами розумного будинку.

## 🛠 Обладнання та Підключення

- **Мікроконтролер**: ESP32 (DevKit V1 або аналог).
- **Драйвер**: Модуль **Mini-L298N H-Bridge** (двоканальний драйвер двигунів).
- **Схема підключення**:
  - **Гірлянда 1 (Канал А/B)**: `GPIO 10`, `GPIO 7` (використовується перший H-міст).
  - **Гірлянда 2 (Резерв/Майбутнє)**: `GPIO 5`, `GPIO 6` (використовується другий H-міст).
  
> **Примітка:** Апаратно передбачено можливість підключення другої незалежної гірлянди до пінів 5 та 6. У майбутніх версіях прошивки планується реалізація повного роздільного керування обома гірляндами.

## 🚀 Встановлення

1. **Встановіть залежності**:
   Переконайтеся, що у вашому середовищі (Arduino IDE / PlatformIO) встановлені бібліотеки:
   - `ESPAsyncWebServer`
   - `AsyncTCP`
   - `ElegantOTA`
   - `WiFiManager` (для ESP32)
   - `AuthenticationMiddleware`
   - ESP32 Board package

2. **Налаштування**:
   Скопіюйте `secrets.h.example` в `secrets.h` та налаштуйте:
   ```cpp
   inline WiFiNetwork wifiNetworks[] = {
     {"YOUR_WIFI_SSID", "YOUR_PASSWORD"},
     {"YOUR_EXT_WIFI_SSID", "YOUR_PASSWORD"} // Опціонально
   };
   
   #define WIFI_HOSTNAME "girlianda"  // Опціонально
   #define AUTH_PASSWORD "your-ota-password"  // Для /update
   ```

3. **Завантаження**:
   Скомпілюйте та завантажте скетч у вашу плату ESP32.

## 📖 Використання

Після запуску відкрийте Serial Monitor (швидкість **115200**).
Ви побачите IP адресу пристрою (наприклад, `192.168.1.105`) або адресу точки доступу (`192.168.4.1`).

Відкрийте цю адресу у браузері для доступу до панелі керування.

### 📡 HTTP API
| Метод | URL | Опис |
|-------|-----|------|
| GET | `/api/status` | Повертає JSON зі станом: `{"mode":1,"speed":50,"brightness":255}` |
| GET | `/api/set?mode=X` | Встановити режим `X` (0-5) |
| GET | `/api/set?speed=X` | Встановити швидкість `X` (1-100) |
| GET | `/api/set?brightness=X`| Встановити яскравість `X` (0-255) |

### 🔐 OTA Оновлення
Для оновлення прошивки через браузер:
1. Відкрийте `http://<IP-адреса>/update` або `http://girlianda.local/update`
2. Введіть пароль (налаштований в `secrets.h`)
3. Виберіть файл прошивки (`.bin`) та натисніть "Update"

### 💻 Serial Моніторинг
Для діагностики підключіть Serial Monitor (швидкість **115200**):
- Відображення процесу підключення до WiFi
- Виведення IP адреси після успішного підключення
- Базові команди для тестування:
  - `m=X` - Встановити режим (0-5)
  - `s=X` - Встановити швидкість (1-100)
  - `b=X` - Встановити яскравість (0-255)
  - `status` - Показати поточний статус
